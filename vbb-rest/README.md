# Cornelicorn Home Assistant Add-on: VBB REST API

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield]

An HTTP REST API for the public transport of Berlin & Brandenburg (VBB), based on
[`vbb-rest`](https://github.com/derhuerst/vbb-rest) by Jannis R.
